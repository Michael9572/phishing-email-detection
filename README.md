# phishing-email-detection
Machine learning project to detect human vs LLM-generated phishing emails using Python, Random Forest, Logistic Regression and Naïve Bayes"
As AI-generated phishing emails become more sophisticated, traditional detection methods are no longer enough. This project addresses that gap by building a classification system that can distinguish between human-written and LLM-generated phishing emails.
 Objectives

Detect and classify phishing emails based on their origin (human vs LLM-generated)
Compare the performance of three machine learning algorithms
Determine the most accurate algorithm for phishing email classification

🛠️ Technologies Used

Python
Random Forest
Logistic Regression
Naïve Bayes
Pandas
Scikit-learn

📊 Results

Successfully classified emails based on accuracy rates
Compared performance across all three algorithms
Identified the best performing algorithm for phishing detection
